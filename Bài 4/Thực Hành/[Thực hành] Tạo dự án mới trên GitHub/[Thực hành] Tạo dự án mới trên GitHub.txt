Lệnh git init dùng để khởi tạo Repository
Lệnh git remote add origin dùng để kết nối tới Remote Reposiotory
Lệnh git add để đưa file đã tạo vào chỉ mục của Git
Lệnh git commit -m "Commit" dùng để lưu lại các thay đổi
Lệnh git push để đẩy các thay đổi ở Local Repository lên trên Remote Repository
